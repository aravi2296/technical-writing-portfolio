# User Guide – Task Management Application

## Introduction

The Task Management Application enables users to create, track, update, and manage daily tasks through a web-based interface.

## Features

- Create new tasks
- Update task status
- Assign task priorities
- Search and filter tasks
- Generate task reports

## System Requirements

- Modern web browser
- Internet connection
- User account

## Logging In

1. Open the application URL.
2. Enter your username and password.
3. Click Login.

## Creating a Task

1. Select New Task.
2. Enter task details.
3. Choose priority level.
4. Click Save.

## Updating a Task

1. Open the task list.
2. Select the task.
3. Modify required fields.
4. Click Update.

## Troubleshooting

### Unable to Login

- Verify username and password.
- Reset password if required.
- Contact system administrator.

## Contact Support

For technical assistance, contact the support team.
